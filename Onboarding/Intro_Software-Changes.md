## Module 3
![alt text](Intro_Software-Changes-img/image-33.png)

Video 1
A substantial part of mhWirth busines is based on software products.
Both new features and delivering upgrades of existing control system solutions.
in order for us to deliver a good customer experience and a software change with the control in quality it requires, we have developed the QA21 Software Changes procedures.


one of the key goals with all procedures, is that they should add value to your daily work. so is the case with the qa21 procedure.
its there for your benefit. to be used whenever you are performing a software change or upgrade for any equipment that are in operation
